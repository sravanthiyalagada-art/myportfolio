# Sravanthi Yalagada — Portfolio Website

🌐 **Live Site:** [sravanthiyalagada-art.github.io](https://sravanthiyalagada-art.github.io) *(update this link after deploying)*

## About

## Features

- Typing animation in the hero section
- Animated skill progress bars on scroll
- Project cards (projects will be added soon)
- Working contact form via [FormSubmit](https://formsubmit.co) — no backend needed
- Fully responsive design for mobile and desktop
- Smooth scroll navigation with active link highlighting

## Getting Started

No installation or build step needed.

```bash
# Clone the repo
git clone https://github.com/sravanthiyalagada-art/sravanthiyalagada-art.github.io.git

# Open in browser
open index.html
```

---

## Deploying on GitHub Pages

1. Go to your repository → **Settings** → **Pages**
2. Under **Source**, select `main` branch and `/ (root)`
3. Click **Save**
4. Your site will be live at `https://<your-username>.github.io`

---

## Customization

| What to change | Where |
|---|---|
| Your name & bio | `index.html` → `#hero` and `#about` sections |
| Skills & percentages | `index.html` → `#skills` section |
| Projects | `index.html` → `#projects` section (replace placeholder cards) |
| Contact email | `index.html` → FormSubmit action URL and contact links |
| Social links | `index.html` → `#contact` section |
| Typing roles | `index.html` → `const roles = [...]` in the `<script>` tag |
